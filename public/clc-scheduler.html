📖 Vercel Deployment & Password Instructions ▼
🌐 How to Deploy to Vercel (Free)
1. Go to vercel.com and sign in (or create a free account).
2. Click Add New → Project.
3. Choose "Deploy without a Git repository" or drag-and-drop.
4. Upload this index.html file (rename CLC_Scheduler.html → index.html).
5. Click Deploy. Your app goes live at a vercel.app URL instantly.
6. To update: go to your project → Deployments → upload a new file.
🔐 How to Change the Default Password
Option A — In the App: Use the "Change Access Password" section above. This saves to this browser only.

Option B — In the HTML file (recommended for shared Vercel deployments):
1. Open CLC_Scheduler.html (or index.html) in Notepad or any text editor.
2. Find this line near the top of the <script> section:
var DEFAULT_PASS = 'CLC-Admin-2024'; 3. Change CLC-Admin-2024 to your desired password.
4. Save the file and re-upload to Vercel.
🔒 Security Notes
• All PHI stays on the local computer (browser localStorage). Nothing is sent to any server.
• The Vercel URL is a blank/empty app — data only appears after a veteran is added locally.
• The password protects access to the app interface. It is not military-grade encryption.
• For stronger access control, consider Vercel Pro ($20/mo) which has built-in password protection at the infrastructure level.
